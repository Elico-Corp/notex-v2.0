NoTex.ch

An online text editor for reStructuredText, Markdown, LaTex and more!

NoTex.ch helps you to write reports using the lightweight markup language reStructuredText aka rST. Once you've written your article or report, it can easily be translated to various formats like PDF, HTML or LaTex. To launch the application, use the following link:

ReStructuredText

Purpose

The rST markup is commonly used for technical documentation, e.g. for software libraries. However, reStructuredText is also very much suitable for other kinds of texts, like articles, reports or theses.

Tutorial

There are many good primers, tutorials and references about rST, e.g. rST Primer is quite a thorough introduction and Quick reStructuredText is a good reference.

Miscellanea

NoTex' front-end has entirely been written with the ExtJS JavaScript and the back-end with the Flask Python library.
